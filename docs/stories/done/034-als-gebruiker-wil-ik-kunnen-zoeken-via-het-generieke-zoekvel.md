# #34 — Als gebruiker wil ik kunnen zoeken via het generieke zoekveld bovenin de topbar.

Als gebruiker wil ik kunnen zoeken via het generieke zoekveld bovenin de topbar. zodat ik snel de juiste gegevens beschikbaar heb.

Er kan gezocht worden op Stalnaam, Eigenaarnaam, paardnaam.

Het zoeken start na het typen van minimaal 2 characters.
Het zoeken start na 500ms na het typen van de laatste letter
Het resultaat wordt getoond in een soort dropdown lijst.
Per getoond item is het duidelijk dmv een icon dat het om een Stal, Paard of Eigenaar gaat.

Bij het aanklikken van de getoonde opties wordt de detailpagina van de Stal, Paard of Eigenaar getoond.

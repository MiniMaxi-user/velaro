# #43 — Als staleigenaar wil ik de werking van het stalfilter uitbreiden

Uitbreiden van dropdown links in de menubar Actieve stal met: "Alle stallen".

Vervolgens worden devolgende paginas gefilterd op de gekozen stal OF alle stallen.

Dashboard
Paarden
Team
Taken

Specifiek Mijn stallen toont altijd ALLE stallen.

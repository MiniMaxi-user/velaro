# #12 — Als ik over mijn naam hover dan verdwijnt de popup en kan ik niet op Uitloggen klikken

Als ik over mijn naam hover dan verdwijnt de popup en kan ik niet op Uitloggen klikken.

Dit komt denk ik doordat je buiten de hoover van je naam komt en nog niet in de popup met je muis zit.
Hierdoor sluit de popup.

Als gebruiker verwacht ik dat ik over mijn naam kan hoveren, de popup getoond wordt met mijn naam erop en daaronder uitloggen (bestaande functionaliteit).

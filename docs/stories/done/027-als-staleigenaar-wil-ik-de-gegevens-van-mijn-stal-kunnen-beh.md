# #27 — Als staleigenaar wil ik de gegevens van mijn stal kunnen beheren

Als staleigenaar wil ik de gegevens van mijn stallen kunnen beheren zodat mijn klanten (paardeigenaren) deze gegevens getoond kunnen krijgen (vervolg story).

Bedenk zelf gegevesn als Naam, adres, postcode, plaats, telefoonnummer, email, factuuradres. Bedenk zelf wat er nog meer nodig is. Zoals openingstijden , website etc.
